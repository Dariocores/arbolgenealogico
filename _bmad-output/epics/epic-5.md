# Epic 5: Configuración y Opciones Avanzadas

## Descripción
Brindar opciones de configuración, como personalización visual, borrado de datos y ajustes avanzados.

## User Stories
- Como usuario, quiero personalizar la apariencia del árbol.
- Como usuario, quiero borrar todos los datos si lo deseo.

## Criterios de Aceptación
- Las opciones deben ser accesibles y seguras.
- Las acciones peligrosas deben requerir confirmación del usuario.
