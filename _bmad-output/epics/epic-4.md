# Epic 4: Exportación e Importación de Datos

## Descripción
Permitir a los usuarios exportar e importar el árbol genealógico en formato JSON para respaldo o migración.

## User Stories
- Como usuario, quiero exportar mi árbol para guardarlo o compartirlo.
- Como usuario, quiero importar un árbol previamente guardado.

## Criterios de Aceptación
- El sistema debe validar el formato de los archivos importados.
- La exportación debe incluir todos los miembros y relaciones.
