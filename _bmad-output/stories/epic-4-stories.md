# Stories para Epic 4: Exportación e Importación de Datos

- Como usuario, quiero exportar mi árbol en formato JSON para guardarlo o compartirlo.
- Como usuario, quiero importar un árbol previamente guardado para continuar mi trabajo.
- Como usuario, quiero recibir alertas si el archivo importado tiene errores de formato.
- Como usuario, quiero asegurarme de que la importación incluya todos los miembros y relaciones.
