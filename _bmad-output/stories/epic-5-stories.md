# Stories para Epic 5: Configuración y Opciones Avanzadas

- Como usuario, quiero personalizar la apariencia del árbol para adaptarlo a mis preferencias.
- Como usuario, quiero borrar todos los datos si lo deseo, con confirmación de seguridad.
- Como usuario, quiero acceder fácilmente a las opciones de configuración.
- Como usuario, quiero restaurar la configuración predeterminada si es necesario.
