# Stories para Epic 3: Visualización Interactiva del Árbol

- Como usuario, quiero ver el árbol de forma clara y atractiva para entender las relaciones familiares.
- Como usuario, quiero interactuar con los nodos para ver detalles o editar miembros.
- Como usuario, quiero que la visualización se actualice en tiempo real al modificar datos.
- Como usuario, quiero navegar fácilmente entre diferentes ramas del árbol.
