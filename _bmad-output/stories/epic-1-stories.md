# Stories para Epic 1: Gestión de Miembros

- Como usuario, quiero añadir un nuevo miembro con nombre y género para construir mi árbol.
- Como usuario, quiero editar la información de un miembro existente para corregir o actualizar datos.
- Como usuario, quiero eliminar miembros del árbol para mantenerlo actualizado.
- Como usuario, quiero recibir mensajes de error si los datos ingresados no son válidos.
